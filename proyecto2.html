Sensor de UltraSonido

El sensor funciona midiendo distancias con un efecto doppler. Una señal va y luego la informacion vuelve. Conectando los cables de esta forma: GND=GND 5V=VCC PIN11=TRIG PIN12=ECHO

Elementos: Arduino UNO, Cable o Jumpers, Sensor ultrasónico HC-SR04, Tabla Protoboard (Opcional)

int trigPin = 11; // Disparador (Trigger) int echoPin = 12; // Eco (Echo) long duration, cm, inches; void setup() { Serial.begin (9600); pinMode(trigPin, OUTPUT); pinMode(echoPin, INPUT); } void loop() { digitalWrite(trigPin, LOW); delayMicroseconds(5); digitalWrite(trigPin, HIGH); delayMicroseconds(10); digitalWrite(trigPin, LOW); pinMode(echoPin, INPUT); duration = pulseIn(echoPin, HIGH); cm = (duration/2) / 29.1; // Divide entre 29.1 o multiplica por 0.0343 inches = (duration/2) / 74; // Divide entre 74 o multiplica por 0.0135 Serial.print(inches); Serial.print("in, "); Serial.print(cm); Serial.print("cm"); Serial.println(); delay(250); }
